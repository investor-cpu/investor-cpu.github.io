Ubuntu18 安装虚拟环境

通过apt安装virtualenv 

sudo apt install virtualenv
sudo apt install virtualenvwrapper
在.bashrc 中添加 下面两行，根据自己的目录进行路径修改 
export WORKON_HOME=$HOME/.virtualenvs
source /usr/share/virtualenvwrapper/virtualenvwrapper.sh
保存后在终端执行以下命令 
source .bashrc
创建指定版本python的虚拟环境 
mkvirtualenv venv --python=/usr/bin/python3.6

Ubuntu18 安装OneNote

只需要两行命令 

sudo apt-get install snap
sudo snap install p3x-onenote

Ubuntu18 安装截图软件

Ubuntu18 安装flameshot

sudo apt install flameshot
配置快捷键： 设置>设备>键盘，设置一个自定义的快捷键（拉倒最下边） 名称：截图 命令填写：flameshot gui 截完图后保存：Ctrl + S 复制到粘贴板：Ctrl + C

Ubuntu18 安装WPS办公软件

卸载原生liboffice

sudo apt-get remove libreoffice-common
以后安装回来使用： 
sudo apt-get install libreoffice

官网下载WPS并安装

下载wps deb包(https://www.wps.cn/product/wpslinux/) 安装 

sudo dpkg -i wps……

第一次打开wps 文档提示缺少字体

下载字体https://pan.baidu.com/s/1mh0lcbY

解压缩

unzip wps_symbol_fonts.zip
sudo mv  wps_symbol_fonts  /usr/share/fonts

修改字体权限，终端执行以下命令

cd /usr/share/fonts/
chmod 755 wps_symbol_fonts
cd /usr/share/fonts/wps_symbol_fonts
chmod 644 *

生成字体索引,终端执行以下命令

cd /usr/share/fonts/wps_symbol_fonts
sudo mkfontscale
sudo mkfontdir

更新字体缓存,终端执行以下命令

sudo fc-cache

重启WPS

参考资料

https://blog.csdn.net/qq_25135655/article/details/104394488

Ubuntu 桌面卡住

PS：Ubuntu桌面卡住，键盘无反应，只能鼠标移动，但是鼠标无法点击 解决方案： 1. 可能等待几分钟，系统会自动反应过来。 2. 注销用户 (1) Ubuntu有6个tty终端，按住Ctrl+Alt+F1可以进入tty1终端，（同理Ctrl+Alt+F2，F3等可以进入其他的tty1终端，这里我们只需要进入一个tty终端就能解决问题）。 (2) 进入tty终端后先输入你的用户名和密码登录。 (3) 执行命令注销桌面重新登陆： 

sudo pkill Xorg    或者 sudo restart  lightdm
(4)执行上述命令后等待两分钟，自然会看到你想要的界面。
